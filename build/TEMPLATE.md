## installation
```shell
npm install -S @jswork/react-searchlight
```

## usage
1. import css
  ```scss
  @import "~@jswork/react-searchlight/dist/style.css";

  // or use sass
  @import "~@jswork/react-searchlight/dist/style.scss";

  // customize your styles:
  $react-searchlight-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
